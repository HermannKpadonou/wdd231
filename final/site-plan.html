Site Plan: Agri-Terroir

Interactive guide to local and heritage crops

Site Name

Name: Agri-Terroir

Reason: This name combines "agriculture" with "terroir", evoking authenticity, origin and local know-how. It's memorable, easy to pronounce and clearly communicates the site's purpose. The name is also SEO-optimized with relevant keywords.

Site Purpose

Agri-Terroir is a platform aiming to:

  • Serve as an interactive guide and encyclopedia for local and heritage crops
  • Provide detailed technical sheets with scientific and practical data
  • Highlight product seasonality and sustainable growing techniques
  • Allow users to create a personalized notebook of favorite crops
  • Promote agricultural heritage and reconnect consumers with their terroir

User Scenarios

Scenario 1: Amateur Gardener

Lucy, 35, looks for information to grow heirloom tomatoes:

  • When to plant? (optimal calendar)
  • How to maintain? (organic techniques)
  • Which local varieties to choose?

Scenario 2: Conscious Consumer

Mark, 42, wants to adapt purchases to seasonality:

  • Which root vegetables are in season in autumn?
  • Where to find these products locally?
  • How to store and cook them?

Scenario 3: Teacher

Sophie, 50, looks for educational resources:

  • Printable technical sheets for students
  • Interactive maps of regional crops
  • History of heritage varieties

Color Scheme

Primary #2E7D32 Text: #FFFFFF

Used for titles, CTA buttons and interactive elements

Secondary #D4A056 Text: #333333

Accents, borders, icons and decorative elements

Background #F5F5F5 Text: #333333

General background for optimal readability

Text #333333 Contrast: 15:1

Body text with excellent accessibility

All combinations verified with WCAG AA/AAA. Minimum contrast ratio of 4.5:1 for normal text.

Typography

Merriweather (Serif)

  • Used for all headings (h1-h6)
  • Base size: 2rem (h1), 1.5rem (h2), 1.2rem (h3)
  • Weight: 700 (bold) for h1-h3, 400 (regular) for h4-h6
  • Line height: 1.3

Lato (Sans-serif)

  • Used for body text, links and interfaces
  • Base size: 1rem (16px)
  • Weight: 400 (regular) for text, 700 (bold) for important elements
  • Line height: 1.6 for comfortable reading
